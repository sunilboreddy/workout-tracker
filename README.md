# 💪 Workout Transformation Tracker

A mobile-friendly workout tracking app built as a single HTML file.

## Features
- 3-Day, 4-Day, and 5-Day workout splits
- Workout logging with exercises, weights, and reps
- Progress tracking (total workouts, this week, best pull-ups, push-ups)
- Progression guides for squats, pull-ups, push-ups
- Works offline, no backend needed
- Data stored in browser localStorage

## How to Use Locally
Just open `index.html` in any browser.

## How to Deploy (GitHub Pages)
1. Push this repo to GitHub (private)
2. Go to Settings → Pages
3. Deploy from main branch
4. Access at: `https://sunilboreddy.github.io/workout-tracker`

## Stack
- Pure HTML, CSS, JavaScript
- No frameworks, no dependencies
- localStorage for data persistence
